Конспект про комбинацию селекторов и специфичность

Типы селекторов CSS

Селекторы - это механизм, который определяет какие стили, указанные в фигурных скобках, будут применены к элементам.

  • Селекторы по элементу. Например, все теги h1/p/table и прочие
  • Селекторы класса. Предназначены для элементов с одинаковым именем класса, размещенных в разных местах сайта. Обозначаются class="______"
  • Селекторы идентификатора. С его помощью можно задавать свойства одному конкретному элементу, с уникальным ID. Обозначаются id="______"
  • Селекторы по атрибуту. С их помощью выбираются элементы, имеющие определенный тип атрибута. Обозначаются [элемент].
  • Универсальный селектор. Соответствует любому элементу, обозначается * {___}
  • Селекторы потомка. Применяются при определенной последовательности вложенности тегов. Например, ul li {_____}
  • Селекторы псевдокласса. Являются дополнением к основному селектору, применяются для более точного обоозначения в зависимости от положения, динамики и пр. Обозначаются селектор:псевдокласс.

Комбинация селекторов

Существует несколько способов объединения селекторов:

  • Группировка - несколько селекторов перечисляются через запятую. Например, h1, h2, h3 и пр.
  • Комбинирование - используется для более точного отбора элементов для форматирования. Перечисляются друг за другом без запятых. Например, a[href] [title]

Специфичность

Все селекторы имеют вес и делятся на четыре основные группы:

  • наименьший вес: селекторы типа и псевдоэлементы
  • малый вес: селекторы класса, атрибута и псевдоклассы
  • средний вес: селекторы ID
  • большой вес: встроенные стили

Если к одному и тому же элементу применяются стили с разным весом, в результате CSS отобразит стиль с наибольшим весом. В случае, если к одному элементу применяются несколько стилей с одинаковым весом, в итоге CSS отобразит тот стиль, который идет последним (то есть находится ближе к концу таблицы стилей).
